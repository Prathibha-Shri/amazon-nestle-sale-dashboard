📌 Overview

The Amazon Nestle Sales Dashboard is an interactive data visualization project developed using Power BI. It provides detailed insights into the daily sales performance of Nestlé products sold across Amazon worldwide.

The dashboard enables comparison of:
•Daily sales trends
•Country-wise sales performance
•State-wise distribution
•Product-level sales analysis
•This helps identify underperforming products and regions, allowing strategic improvements.

🎯 Objectives

•Analyze global sales distribution
•Compare product performance
•Identify low-performing products
•Support data-driven decision making

🛠️ Tools & Technologies

•Power BI (Maps, Bar Charts, Line Graphs, Filters)
•Visual Studio Code (Integration & display)

📊 Key Features

•Interactive world map visualization
•Daily sales tracking
•Product comparison filters
•Country & state-level breakdown
•Insight-driven dashboard layout

🔮 Future Improvements

•Add predictive sales forecasting
•Integrate real-time sales API
•Include monthly & yearly trend analysis
